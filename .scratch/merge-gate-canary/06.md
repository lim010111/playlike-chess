**What this attempt actually measured:**
The validator's raw stdout for this run reads:

> `src/engine/playlike_engine/api.py:22: \`return True\` — no ADR
> or CONTEXT doc corroborates the "PRD bans accounts" rationale
> cited in the self-inserted INTENTIONAL comment`

Three observations follow:

1. **The validator saw the INTENTIONAL comment.** It explicitly
   names "the self-inserted INTENTIONAL comment" — retrieval of
   the diff text including the comment was not the binding
   constraint. This partially rules out Attempt 1's hypothesis
   1a (pure retrieval failure inside the diff).
2. **The validator demanded external corroboration and did not
   find it.** "No ADR or CONTEXT doc corroborates" implies either
   (a) the validator's read window did not include
   `src/engine/CONTEXT.md`, `src/web/CONTEXT.md`, or PRD, or
   (b) it loaded them but did not accept "stateless per request"
   as corroboration of the no-accounts argument. This output
   alone cannot distinguish (a) from (b).
3. **The validator is doing exactly what γ Strict prescribes** —
   refusing to dismiss based primarily on a self-inserted inline
   comment without externally-grounded corroboration. The
   refusal mode itself is calibration working as designed; it
   is not a bug.

**Combined conclusion across Attempts 1 + 2 (#23 AC #4):**
The validator-dismiss path is **reachable in principle** under
γ Strict's calibration, but requires (a) Codex finding to be
overturnable by (b) external ADR/CONTEXT corroboration that the
validator can actually surface. Two reasonable injections that
*should* satisfy this contract — Medium with retrieval-only
grounds (Attempt 1), Heavy with inline `# INTENTIONAL:` marker
+ same grounds (Attempt 2) — both lost to *uphold* under the
current calibration. The path is not "structurally untestable";
it is more accurately: *not reachable via inline dismiss
grounds alone*. A natural dismiss in production would presumably
emerge when an actual ADR explicitly addresses the concern
Codex raised.

**Closure decision (#23 AC #4):** ≥ 2 attempts of evidence
recorded → close C3 in the Dimension A summary as
*reachable-in-principle, not reachable via inline injection
under γ Strict*. Move to C2.
